# A/B Testing Analysis

## 📊 Project Overview

This project analyzes an A/B experiment to evaluate the impact of changes on user behavior and overall performance metrics.

The objective was to determine whether the experimental variation produced statistically significant improvements compared to the control group.

---

## 🛠 Tools & Technologies

- Python (Pandas, NumPy)
- Statistical Analysis
- Hypothesis Testing
- A/B Testing
- Data Cleaning & Preprocessing
- Data Visualization (Matplotlib, Seaborn)

---

## 🔎 Key Steps

- Cleaned and prepared experimental data
- Defined null and alternative hypotheses
- Calculated conversion rates for control and test groups
- Performed statistical significance testing
- Interpreted results to support decision-making

---

## 📈 Results

The analysis determined whether the variation led to a statistically significant performance improvement and provided recommendations based on quantitative evidence.
